# Grants & Compute Credits

Researched April 2026. Re-check deadlines before applying.

## Act Now (deadline < 6 weeks)

None currently — Amazon Research Awards (May 6, 2026) requires institutional PI affiliation; ineligible as an independent developer.

## Cash Grants (rolling)

| Program | Value | Notes |
|---|---|---|
| [AI Grant](https://aigrant.org/) | $5K–$50K cash + $20K GPU credits | No equity, no IP claim; only ask is a 90-day blog post. Best fit for independent devs. **Note: program may be dormant — verify via @aigrant_ on X before submitting.** Draft at `applications/ai_grant.md`. |
| [a16z Open Source AI Grants](https://a16z.com/supporting-the-open-source-ai-community/) | undisclosed | Explicitly funds "optimizing models for low-powered machines" and "novel inference tooling" — exact match. |
| [NSF PESOSE — AI Agent Ecosystems](https://www.nsf.gov/funding/opportunities/dcl-advancing-artificial-intelligence-ai-agent-ecosystems) (NSF 26-506) | $300K (Phase I) / $750K (Phase II) | Institutional affiliation likely required. Agentic protocols + open source. |

## Compute Credits (rolling)

| Provider | Amount | Accessible? | Notes |
|---|---|---|---|
| [Modal Startups](https://modal.com/startups) | up to $25K | Ask — VC-gated by default | Email startups@modal.com for OSS/independent exception. Draft at `applications/modal_startups.md`. |
| [CoreWeave Startup Accelerator](https://www.coreweave.com/blog/coreweave-launches-startup-accelerator-program-democratizing-access-to-gpu-compute-in-the-cloud) | negotiated | Yes | No affiliation requirement; credits negotiated, not published. Expect pitch toward compute-for-equity — decline if offered. Draft at `applications/coreweave.md`. |
| [Google TPU Research Cloud](https://sites.research.google/trc/about/) | 1000s of TPU-hours | Yes | Open application. Must publish/open-source. Caveat: requires JAX or PyTorch/XLA — non-trivial stack migration. Draft at `applications/google_trc.md`. |
| [Lambda Labs Research Grant](https://aicredits.dev/submissions/44-lambda-labs-research-grant-up-to-5-000) | up to $5K | **No** | Explicitly requires institutional affiliation — "not for indie hackers." Skip. |
| [AWS Activate Founders](https://aws.amazon.com/activate/) | $1K | Yes (low value) | Self-service, no VC needed. $300K figure is Portfolio tier (requires VC referral) — not accessible. Not worth the effort given Modal-first stack. |
| [AWS Cloud Credits for Research](https://pages.awscloud.com/aws-cloud-credit-for-research.html) | $5K–$250K+ | **No** | Faculty affiliation required. |
