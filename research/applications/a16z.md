## Email / application text

**Subject:** [grant inquiry] A highly specialized, open-source coding agent — local, private, free

I was a senior engineer at Bionic (raised $350M Series D, acquired by CrowdStrike), working on large-scale security intelligence systems. I'm now building Merlin.

The vision: every developer has a fast, private, always-on coding agent running locally on their MacBook — no API key, no cloud dependency, zero marginal cost.

Merlin is a highly specialized 3B open-source LLM built exclusively for agentic coding. Not a general assistant — a brute-force execution layer. A frontier model (Claude, GPT-4) plans; Merlin executes: grep, file edits, bash, test runs — in parallel, at zero cost. The natural integration: Claude Code as orchestrator, Merlin as the low-level worker handling everything that doesn't require frontier reasoning.

Two deployment modes:
- **Local** — runs on any M-series MacBook, private, always-on, free
- **Hosted** — 100–1000 parallel workers via GPU batching, pay-per-task

This two-tier model cuts API costs 50%+ for power users and unlocks parallel repo-wide operations (rename across 200 files in seconds) that are impractical with a single frontier model call.

What it is:
- 3B parameter model (7B to follow), int4 quantized, ~1.5GB, >500 tok/s on M3
- Pre-trained from scratch on 100B tokens — code, bash, agentic execution traces
- Post-trained with RL on verifiable bash/filesystem task rewards
- MIT license — weights, training code, data pipeline, all open

Compute ask: ~$5K for the 3B run, ~$35K if we scale to 7B. Open to talking.
