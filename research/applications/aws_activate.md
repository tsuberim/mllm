# AWS Activate

Apply at: https://aws.amazon.com/startups/lp/aws-activate-credits
Value: up to $300K credits (new 2026 AI tier for LLM training costs)
Rolling. Startup program — no institutional affiliation needed, but typically expects a company entity or startup.

**Gotcha:** AWS Activate is designed for startups, not individual developers. Credits are tied to an AWS account; may require a business entity. Worth applying as a sole proprietor if you have one, or using a startup accelerator/incubator as a path in.

**Alternative path:** Some accelerators (YC, Antler, etc.) automatically unlock AWS Activate for their companies. If you're considering applying to an accelerator anyway, this comes bundled.

## Application text

**Company/project name:** Merlin (open-source LLM project)

**Description:**

Building an open-source 3B language model for agentic coding — pre-trained from scratch on 100B tokens, post-trained with RL on verifiable rewards, targeting local inference on Apple Silicon. MIT licensed.

**Stage:** Pre-revenue, bootstrapped. ML infrastructure is operational (data pipeline, training loop, eval harness).

**AWS usage plan:** Pre-training on p4de.24xlarge or similar H100 instances (~1,520 GPU-hours), data pipeline on CPU instances, model hosting for public API endpoint post-launch.

**Estimated AWS spend (12 months):** $10K–$15K for training compute; $2–5K for inference hosting of public model endpoint.
